# 阿里轻量云部署指南

本文档记录 FastAPI + SQLite + 本地文件工作区版本的部署方式。代码放在 GitHub，运行数据放在服务器的 `/data/ProjectWorkspace`，不要把 `app.sqlite`、PDF、项目文件提交到 GitHub。

## 1. 服务器准备

推荐镜像：Ubuntu 22.04 或 24.04。

在阿里云轻量应用服务器控制台开放防火墙端口：

```text
22    SSH
80    HTTP
443   HTTPS
```

登录服务器：

```bash
ssh root@your-server-ip
```

安装基础依赖：

```bash
apt update
apt install -y python3 python3-venv python3-pip git nginx
```

## 2. 拉取代码

```bash
mkdir -p /opt
cd /opt
git clone https://github.com/silicontalks/cihlab-wiki-v1.git
cd /opt/cihlab-wiki-v1/backend
```

创建虚拟环境并安装依赖：

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## 3. 准备数据目录

```bash
mkdir -p /data/ProjectWorkspace
```

应用通过环境变量读取工作区路径：

```text
PROJECT_WORKSPACE_PATH=/data/ProjectWorkspace
```

该目录会保存：

```text
/data/ProjectWorkspace/app.sqlite
/data/ProjectWorkspace/projects/
/data/ProjectWorkspace/backups/
/data/ProjectWorkspace/attachments/
```

## 4. 手动启动测试

```bash
cd /opt/cihlab-wiki-v1/backend
source .venv/bin/activate
PROJECT_WORKSPACE_PATH=/data/ProjectWorkspace \
  uvicorn app.main:app --host 0.0.0.0 --port 8000
```

测试：

```text
http://your-server-ip:8000/api/health
```

预期返回：

```json
{"status":"ok"}
```

## 5. systemd 服务

创建 `/etc/systemd/system/cihlab-wiki.service`：

```ini
[Unit]
Description=CIHLab Wiki FastAPI App
After=network.target

[Service]
WorkingDirectory=/opt/cihlab-wiki-v1/backend
Environment=PROJECT_WORKSPACE_PATH=/data/ProjectWorkspace
ExecStart=/opt/cihlab-wiki-v1/backend/.venv/bin/uvicorn app.main:app --host 127.0.0.1 --port 8000
Restart=always
RestartSec=3

[Install]
WantedBy=multi-user.target
```

启用并启动：

```bash
systemctl daemon-reload
systemctl enable cihlab-wiki
systemctl start cihlab-wiki
systemctl status cihlab-wiki
```

查看日志：

```bash
journalctl -u cihlab-wiki -f
```

## 6. Nginx 反向代理

创建 `/etc/nginx/sites-available/cihlab-wiki`：

```nginx
server {
    listen 80;
    server_name your-domain-or-server-ip;

    client_max_body_size 100M;

    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

启用配置：

```bash
ln -s /etc/nginx/sites-available/cihlab-wiki /etc/nginx/sites-enabled/cihlab-wiki
nginx -t
systemctl reload nginx
```

## 7. HTTPS

如果已经有域名并完成解析，可以安装证书：

```bash
apt install -y certbot python3-certbot-nginx
certbot --nginx
```

中国大陆服务器绑定域名通常需要先完成备案。没有备案时，可以先用服务器 IP 访问，或使用中国香港等不需要大陆备案的地域。

## 8. 更新部署

```bash
cd /opt/cihlab-wiki-v1
git pull
cd backend
source .venv/bin/activate
pip install -r requirements.txt
systemctl restart cihlab-wiki
```

## 9. 备份建议

至少备份以下目录：

```text
/data/ProjectWorkspace/app.sqlite
/data/ProjectWorkspace/projects/
/data/ProjectWorkspace/backups/
```

建议同时使用：

- 应用内备份：`/api/workspace/backup`
- 服务器快照或数据盘快照
- 定期把 `/data/ProjectWorkspace` 同步到另一台机器或对象存储
